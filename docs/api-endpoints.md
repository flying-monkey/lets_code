# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `GET /api/users`
- `POST /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Cities

- `GET /api/cities`

### Registrations

- `GET /api/registrations`
- `POST /api/registrations`

### Events

- `GET /api/events`
- `POST /api/events`
