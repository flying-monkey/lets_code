# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Cities

- `GET /api/cities`

### Hosts

- `GET /api/cities/id/host/id`

### Events

- `GET /api/events`
- `POST /api/events`
